# Bank Management System (Java)

## Description
A console-based banking application developed using Java. It allows users to create accounts and perform basic banking operations.

## Features
- Account Creation
- Deposit Money
- Withdraw Money
- Check Balance

## Tech Stack
- Java
- OOP Concepts
- ArrayList

## How to Run
1. Compile:
   javac Main.java Account.java Bank.java

2. Run:
   java Main
